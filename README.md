## üöÄ Getting Started

### Prerequisites

- Python 3.10 or higher
- `pip` (Python package manager)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/krashitos/propvision-ai-real-estate-enhancer.git
   cd propvision-ai-real-estate-enhancer
   ```

2. **Set up a Virtual Environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

### Running the App

```bash
python app.py
```

Visit: **[http://localhost:8000](http://localhost:8000)**

## üìñ Usage

### Image Enhancement
1. Drop or upload a property image in the **Enhance Image** tab.
2. Click **Analyze & Enhance** ‚Äî AI scores and diagnoses the image.
3. Click **Generate Enhanced Version** ‚Äî AI creates a professional listing photo.
4. Use the **Before/After slider** to compare results.
5. Click **Download** to save the enhanced image.

### Virtual Staging
1. Switch to the **Virtual Staging** tab.
2. Select a **room type** (Living Room, Kitchen, etc.) and **design style** (Modern, Scandinavian, etc.).
3. Click **Generate Staged Room** ‚Äî AI creates a photo-realistic staged room.
4. Download or regenerate as needed.
